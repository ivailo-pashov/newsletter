There Is Always More to Learn

 
 
 
 
 
 

Functional Programming Patterns with Java 8​

Amazing hands on coding session demonstrating best practices, patterns and anti-patterns with Lambdas and Streams by Victor Rentea, which includes:
Bloated Lambdas, A Happy Predicate, Stream Wrecks, How to Kill The Biggest Bug, Monads, Passing-a-block, The Loan Pattern, and 5 Ways to Implement Type-Specific Logic.
Make sure not to miss the video!

 
 
 
 
 
 

Java Security Animated

Java security has been inherently a complex topic ever since the introduction of the security sandbox model in JDK 1.2 and the evolution of security APIs provided by the JDK.
You will find the Java Security Animated application a great walkthrough if you are newcomer to the security world of the JDK as well as reference if you are more experienced with it.

 
 
 
 
 
 

Lambdas are not Functional Programming

Just because you are using Lambda expressions, does not mean you are doing functional programming.
This article brings more insights about what functional programming really is and how it is not enforeced by the Java compiler.

 
 
 
 

Introduction to Modern Network Load Balancing and Proxying

Load balancing is one of the core concepts required for building reliable distributed systems.
However, there is a lot of confusion on what it really is and most of the related information is not up-to-date. Take a look at this article to get a great overview about all the related concepts!

 
 
 
 

Architectures that Bend but Don't Break​

Matt Stine discusses the architecture of robust systems which are adapting to changing conditions in order to not only survive stress but sometimes to benefit from it.

 
 
 
 

14 High-Performance Java Persistence Tips​ 

A high-performance data access layer requires a lot of knowledge about database internals, JDBC, JPA, Hibernate, and this post summarizes some of the most important techniques you can use to optimize your enterprise application.

 
 
 
 
 
 
 
 

How to Use the Spring Config Server​

A very nice tutorial how you can get started with Spring Cloud Config to create remote configurations in your applications. The demo is based on bootstrapping the configuration server with a Git repo and starting the corresponding clients.

 
 
 
 

How Fast is Spring?​

In 2018 we have seen the rise of Java Microframeworks which are much more efficient compared to Spring. It's time to see the response from Spring!
Check the article to find out what optimizations and tweaks are provided by Spring and Spring boot to improve the startup time and heap usage of your applications.

 
 
 
 
 

How Low Can You Go?
Ultra Low Latency Java in the Real World

An exciting talk about ultra low latency programming. You will get a great overview of how Java is suitable for low latency and what development techniques could be employed to achieve this as well how you could adopt in the distributed world of Microservices.  

 
 
 
 
 
 

Unit Testing Anti-Patterns​

Building a maintainable codebase also means you should also take care for high quality unit tests.
However, sometimes you can accidentally introduce anti-patterns in your tests. Take a look if you are not applying some of them with the help of this comprehensive list of anti-patterns! 

 
 
 
 
 
 

Detecting Memory Leaks From a JVM Heap Dump​

Yet another cool article how you can troubleshoot memory leaks.

Dealing with concurrency is not very easy even for experienced developers.

Java 8 Stream Collectors groupingBy Example

Similar to SQL GROUP BY Java 8 Collectors API provides a groupingBy method which is a very useful aggregation function.
