Java Proposal Would Spot Data Races
With a Thread Sanitizer 

A new OpenJDK proposition has been made based on the work at Google. Called the Atlantis Project, the effort would explore possibilities for integrating HotSpot and the JVM tool interface with performance monitoring and analysis capabilities. Key to the proposal is evaluating and incubating a thread sanitizer, which could provide a dynamic data-race detector for Java and JNI code.

Infer - a Static Analysis Tool 

Infer is a great tool open sourced by Facebook which will help you detect bugs and make your code more maintainable! It has the following checkers:
   AI - checks based on Abstract Interpretation for ThreadSafety, AnnotationReachability and immutable casts.
   Eradicate - type checker for @Nullable annotations
   RacerD - search for a high-confidence class of data races
