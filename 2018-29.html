Apache Camel 2.22 Released with
Spring Boot 2 Support

The latest Apache Camel 2.22.0 release is here! It is the first release to officially bring you full Spring Boot 2 support. The release also includes some other noteworthy improvements such as an internal optimisation to the toD EIP with HTTP endpoints to better reuse existing http connections when sending to the same host with all sort of dynamic parameters.
