Whats New for Spring Boot in IntelliJ IDEA 2018.3 

A new version of IntelliJ IDEA is available for download! The latest version of Spring Boot is now fully supported. Changes include enhanced application configuration, better plugins integration when using Spring Initializr, initial support for Spring Cloud Stream and faster performance on startup and IDE model calculations.

 
 
 
 
 
 
 

Gradle 5.0 Milestone 1

Gradle 5.0 offers you some great new features and improvements. You can, for example, migrate your build logic from Groovy to Kotlin using Gradle Kotlin DSL, specify a timeout duration for a task or configure different modules belonging to the same logical group to have identical versions in a dependency graph. Do not miss the article for the complete list of all the new and noteworthy improvements.
