How To Optimize JPQL And Criteria API Query Plans With Hibernate Statistics 

Every entity query, be it JPQL or Criteria API, needs to be parsed and compiled to an Abstract Syntax Tree in order to generate the associated SQL query. The entity query compilation takes time so Hibernate provides a QueryPlanCache to store already-compiled plans. This article will show you how to monitor this cache and speed up your IN query performance using Statistics object.
