The latest JEP targeted to JDK 11 is here. Although being experimental, its goal  is to make Java a more attractive platform by drastically reducing the length of GC pauses for TB large heaps. For more information dont miss to check this video too!

 
 

Time to Upgrade

 
 
 

IntelliJ IDEA 2018.2 EAP3:
Advanced @Contract Annotations and More

A new Early Access Program build of IntelliJ IDEA 2018.2 is assembled. It offers improved @Contract annotation, enhanced breakpoint intentions that allow filtering by a caller method and more.

 
 
 
 
 
 
 

Gradle Releases Version 4.7 with Support for Java 10

Gradle 4.7 ships with Java 10 runtime support, incremental annotation processing, Kotlin DSL 0.16 support and more. Check out the article to see the full change list.

 
 
 
 
 
 
 

Lombok 1.18.0

Lombok can help you reduce the amount of boilerplate code. The latest version includes full Java 10 support, speedup improvements with Gradle incremental builds, and many minor improvements.
