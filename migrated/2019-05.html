Reactive HTTP/2 Requests And Responses In Java 11

With Java 11s new HTTP API you can do more than just HTTP/2 and asynchronous requests you can throttle, you can stream (to conserve memory), and you can expose a result as soon as you found it (instead of waiting for the entire body to arrive). Follow the article and its demo to learn more about reactive streams, how Javas new HTTP/2 API uses it and how streaming request and response bodies works.

Progressive Delivery in Kubernetes: Blue-Green and Canary Deployments

Progressive Delivery is the next step after Continuous Delivery, where new versions are deployed to a subset of users and are evaluated in terms of correctness and performance before rolling them to the totality of the users and rolled back if not matching some key metrics. In this article you are going to look at three of projects which make it easier with Kubernetes: Shipper, Istio and Flagger.
