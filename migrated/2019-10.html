Teeing Collector in Java 12

Java 12 comes out in two weeks and, with switch expressions, takes the first step towards pattern matching. But the new release has more to offer than that well, a little bit more. It also introduces the "teeing collector" as a static method Collectors::teeing. Just like the linux command tee, this collector forwards its input to two other collectors before merging their results with a function.
Easy enough, but read on if you want to learn more.
