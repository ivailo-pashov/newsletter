How to Escape SQL Reserved Keywords With JPA and Hibernate 

Because SQL is a declarative language, the keywords that form the grammar of the language are reserved for internal use, and they cannot be employed when defining a database identifier. In this article, however, you are going to see how you can escape SQL reserved keywords with JPA and Hibernate and use the reserved words for your own use.

 
 
 
 
 
 

Why, When and How to Use DTO Projections With JPA and Hibernate 

DTOs are easy to use and the most efficient projection for read-only operations. So, whenever you dont need to change the requested information, you should prefer a DTO projection. But what exactly is a DTO? How does such a projection work with Hibernate? And how can you use it in your queries? You will get answers to all these questions in this article and find a way to use a DTO projection that doesnt require a custom DTO class.

You will also learn how this performance can be drastically improved with the help of Kotlin coroutines when they are used with the correct data structures.

 
 
 
 
 
 

All You Need To Know About Testing Web Controllers with Spring Boot

Otherwise, we may be in for ugly surprises at runtime. Find out how you can test your application more thoroughly in this article!
