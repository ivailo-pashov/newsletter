Apache Dubbo

The Apache Software Foundation has promoted its high-performance Java-based remote procedure call (RPC) framework, Apache Dubbo, from incubator to top-level project status. Dubbo includes interface-based remote call, fault tolerance, intelligent load balancing, and automatic service registration and discovery.  Apache
In this article, you are going to find an overview of it and what makes it so popular. You can also check this quick start guide to find out all the relevant information.
