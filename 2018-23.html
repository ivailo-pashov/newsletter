HeapHero - Java & Android Heap Dump Analyzer

Analyze your heapdumps online with this amazing tool! It will help you discover memory leaks, regressions in memory usage and it also provides tips to reduce the overall memory usage. It is totally free so just give it a try!

 
 
 
 
 
 
 

Apache Pulsar 2.0

The new version of Apache Pulsar offers several new features and major improvements, namely Pulsar functions which represent a lightweight compute framework, new type-safe Java API for producer/consumers, schema registry which enforces schema on topics, Topic compaction and much more.
