Spring Boot: Creating Asynchronous Methods Using @Async Annotation 

Sometimes you want to execute slow I/O bound tasks in asynchronous way but this often involves a lot of work dealing with setting up appropriate ExecutorService. Spring allows you to build an abstraction over the underlying executing thread pool being used with @Async annotation. You can use it in two ways:
      Fire-and-forget (void methods)
      With returned result for parallel tasks (Future and CompletableFuture)
See how you can use the second approach in this article!
